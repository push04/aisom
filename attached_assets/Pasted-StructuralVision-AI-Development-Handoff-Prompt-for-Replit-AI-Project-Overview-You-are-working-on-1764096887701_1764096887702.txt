Project Overview
You are working on StructuralVision AI, a production-ready React web application for structural engineering analysis. The app combines AI + Engineering with TensorFlow.js for computer vision and OpenRouter for LLM-based explanations.

Current Tech Stack:

Frontend: React + Vite + Tailwind CSS + JavaScript
AI/ML: TensorFlow.js (@tensorflow-models/pose-detection)
Backend: Netlify Functions (OpenRouter proxy, Geocoding proxy)
Charts: Recharts
Maps: OpenStreetMap (iframe embed)
Deployment: Netlify (configured)
